# 위필링 앱 콘텐츠 등급 분석 및 권장사항

## 앱 콘텐츠 분석

### 1. 주요 기능 분석

#### 🔍 소셜 기능
- **게시판**: 텍스트, 이미지 게시 가능
- **댓글 시스템**: 원댓글, 대댓글 지원
- **좋아요 기능**: 게시글 및 댓글 좋아요
- **친구 시스템**: 친구 요청, 수락, 카테고리별 관리
- **직접 메시지**: 1:1 개인 메시지 기능

#### 🎭 익명성 기능
- **익명 게시**: 사용자 선택에 따른 익명 게시 가능
- **공개 범위 설정**: 전체 공개, 친구 공개, 카테고리별 공개
- **익명 댓글**: 댓글도 익명으로 작성 가능

#### 👥 모임 기능
- **모임 생성**: 스터디, 식사, 카페, 문화 활동 등
- **참여 관리**: 모임 참여 신청 및 관리
- **모임 채팅**: 참여자 간 그룹 채팅
- **리뷰 시스템**: 모임 후 참여자 평가

#### 🛡️ 안전 장치
- **학교 인증**: 대학 이메일 인증 필수
- **신고 기능**: 부적절한 콘텐츠 신고 가능
- **차단 기능**: 특정 사용자 차단
- **콘텐츠 필터링**: 부적절한 언어 필터링 (기본 구현)

### 2. 위험 요소 분석

#### ⚠️ 잠재적 위험 요소
1. **익명 게시**: 악용 가능성 (사이버 불링, 허위 정보)
2. **직접 메시지**: 개인간 소통으로 인한 괴롭힘 가능성
3. **사용자 생성 콘텐츠**: 부적절한 텍스트/이미지 업로드 가능
4. **모임 기능**: 오프라인 만남으로 인한 안전 문제

#### ✅ 완화 조치
1. **학교 인증 시스템**: 신원 확인 가능
2. **신고/차단 시스템**: 부적절한 행동 대응
3. **콘텐츠 필터링**: 기본적인 욕설 필터링
4. **관리자 모니터링**: 신고 접수 시 검토 시스템

## Google Play 콘텐츠 등급 권장사항

### 🎯 권장 등급: **청소년 이용불가 (Mature 17+)**

#### 근거:
1. **타겟 사용자**: 만 18세 이상 대학생
2. **익명 소통**: 익명 게시 및 댓글 기능
3. **직접 메시지**: 사용자 간 개인 메시지 교환
4. **사용자 생성 콘텐츠**: 필터링되지 않은 자유로운 콘텐츠 생성
5. **오프라인 만남**: 모임 기능을 통한 실제 만남 주선

### 📋 Google Play 콘텐츠 등급 설문 예상 답변

#### 1. 폭력성 (Violence)
- **답변**: None (없음)
- **이유**: 폭력적 콘텐츠 없음

#### 2. 성적 콘텐츠 (Sexual Content)
- **답변**: None (없음)
- **이유**: 성적 콘텐츠 기능 없음

#### 3. 약물/알코올 (Drugs & Alcohol)
- **답변**: None (없음)
- **이유**: 관련 기능 없음

#### 4. 사용자 상호작용 (User Interaction)
- **답변**: **Yes - Users can communicate with each other**
- **세부사항**:
  - ✅ 사용자 간 메시지 교환 가능
  - ✅ 게시글 및 댓글 작성 가능
  - ✅ 개인정보 공유 가능 (프로필)
  - ✅ 위치 정보 공유 가능 (모임 장소)

#### 5. 사용자 생성 콘텐츠 (User Generated Content)
- **답변**: **Yes - Users can create content**
- **세부사항**:
  - ✅ 텍스트 게시글 작성
  - ✅ 이미지 업로드
  - ✅ 댓글 작성
  - ✅ 프로필 정보 입력

#### 6. 개인정보 수집 (Personal Information)
- **답변**: **Yes - App collects personal information**
- **세부사항**:
  - ✅ 이메일 주소 (학교 인증용)
  - ✅ 닉네임
  - ✅ 프로필 사진
  - ✅ 학과/학번 정보

#### 7. 위치 정보 (Location)
- **답변**: **No** (현재 GPS 기능 없음)
- **참고**: 모임 장소는 텍스트로만 입력

### 🏷️ 카테고리 권장사항

#### 주 카테고리: **소셜 (Social)**
#### 부 카테고리: **교육 (Education)**

### 📝 추가 권장사항

#### 1. 콘텐츠 정책 강화
```
- 더 강력한 콘텐츠 필터링 시스템 도입
- 신고 처리 프로세스 문서화
- 커뮤니티 가이드라인 명시
```

#### 2. 안전 기능 추가 고려
```
- 모임 안전 가이드라인 제공
- 의심스러운 활동 자동 감지
- 부모/보호자 알림 기능 (선택사항)
```

#### 3. 개인정보 보호 강화
```
- 개인정보 처리방침 앱 내 쉽게 접근 가능하도록
- 데이터 삭제 요청 기능 강화
- 프라이버시 설정 옵션 확대
```

## 결론

위필링은 대학생을 대상으로 한 건전한 소셜 플랫폼이지만, 익명 기능과 사용자 간 직접 소통 기능으로 인해 **청소년 이용불가 (Mature 17+)** 등급을 받을 가능성이 높습니다. 

이는 앱의 타겟 사용자층(만 18세 이상 대학생)과 일치하므로 문제없으며, 오히려 적절한 연령 제한을 통해 더 안전한 커뮤니티를 구축할 수 있습니다.
