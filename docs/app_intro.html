카테고리로 관계를 나누고,
부담 없이 연결되는 SNS

Wefilling은 ‘학교/팀플/동아리/친구/가족’처럼 관계의 맥락을 카테고리로 정리하고, 그 맥락에 맞게 프로필과 공개 범위를 선택할 수 있도록 설계된 소셜 네트워킹 플랫폼입니다.

기능 한눈에 보기 확장 로드맵 Google Play(준비중) App Store(준비중)
3~4명 스몰 모임 중심으로 ‘첫 만남’ 부담을 낮춤
학교 메일 인증 + 국적 표시로 신뢰 기반 연결
관계 카테고리 기반으로 게시글 공개 범위를 명확히

Wefilling이 해결하려는 것

“애매한 관계”에서 말 걸기·공유하기가 어려운 구조

기존 SNS는 이미 친한 관계 중심으로 굳어지거나, 익명 커뮤니티는 신뢰 기반 연결이 어렵습니다. Wefilling은 관계의 맥락을 먼저 정리해, 연결과 공유가 자연스럽게 이어지도록 설계했습니다.

현재 운영 방향

  • 서비스 안정화와 사용자 경험(UX) 개선
  • 관계/행동 데이터를 기반으로 추천·광고 효율을 높일 준비
  • 캠퍼스 기반 확산 → 다문화 사용자 데이터 축적 → 해외 대학 확장

왜 Wefilling인가

관계의 맥락이 없어서 생기는 피로

같은 사람이더라도 ‘팀플에서의 나’, ‘동아리에서의 나’, ‘친한 친구 앞의 나’는 다릅니다. 그런데 하나의 피드와 하나의 정체성으로만 소통하면, 공유 전부터 신경 쓸 것이 많아지고 결국 “아무 것도 안 올리는” 방향으로 굳어집니다. Wefilling은 이 구조를 ‘카테고리’로 해결합니다.

어떻게 작동하나

카테고리 → 공개범위 → 연결

1) 관계를 카테고리로 정리

학교/팀플/동아리/친구 등 맥락을 먼저 분리해 관계 관리를 단순화합니다.

2) 공개범위를 맥락에 맞게

게시글·모임을 올릴 때 ‘누구에게 보여줄지’를 자연스럽게 선택할 수 있습니다.

3) 작은 연결부터 시작

3~4명 스몰 모임 구조로 첫 만남의 심리적 장벽을 낮춥니다.

핵심 기능

실사용 흐름 중심

Friends Category

관계 카테고리 관리로, 같은 플랫폼 안에서 ‘관계 맥락’을 유지합니다.

게시글 · 공개범위 설정

부담을 줄이는 공개범위 선택으로, 잡다한 이야기·질문·도움 요청이 자연스러워집니다.

Direct Message

친구/익명 DM 흐름으로, 게시글에서 대화로 이어지는 전환을 지원합니다.

Meeting

스터디·식사·카페·문화 등 목적 기반 소규모 모임을 빠르게 만들고 참여합니다.

Meeting Review

‘기록 + 공유’로 모임 경험이 축적되고, 관계가 끊기지 않게 이어집니다.

My Profile

닉네임·국적·한 줄 소개, 그리고 언어 선택(한국어/영어)로 다문화 사용성을 강화합니다.

Notification

관계 카테고리와 흐름에 맞춘 알림 설계로 중요한 소통이 묻히지 않게 합니다.

Advertisement

사용 경험을 해치지 않는 배치와 맥락 기반 타게팅을 전제로 광고 수익을 확장합니다.

로드맵(운영 과제)

스토어 등록, 안정성 테스트 등 정식 배포 완성도를 단계적으로 끌어올립니다.

신뢰 기반 커뮤니티 설계

학교 인증과 정보 표시

Wefilling은 “누구나 들어올 수 있는 익명 공간”이 아니라, 캠퍼스 기반의 신뢰를 바탕으로 확산되는 구조를 지향합니다. 학교 메일 인증과 국적 표시를 통해 연결의 불확실성을 낮추고, 다문화 환경에서도 안전하게 소통할 수 있는 기본 조건을 제공합니다.

확장 전략

캠퍼스 → 다문화 → 해외 대학

초기에는 ERICA 중심으로 사용자 밀도와 활동성을 확보하고, 다문화 사용자 데이터가 자연스럽게 축적되는 환경에서 서비스의 ‘관계 맥락’ 설계를 고도화합니다. 이후에는 대학 이메일 도메인 기반 확장 구조로 해외 대학으로 진입하는 방식의 검증 중심 확장을 추진합니다.

비즈니스 모델

광고 중심 · 확장형 수익 구조

Wefilling은 유료 구독을 핵심 수익 구조로 두지 않고, 무료 이용을 기본으로 사용자 기반과 데이터 밀도를 먼저 확보합니다. 핵심 수익원은 관계 카테고리, 콘텐츠 유형, 모임 주제, 지역·국적 정보 등 ‘맥락 데이터’를 활용한 광고 모델이며, 서비스 확장(캠퍼스→글로벌)이 곧 광고 시장 확장으로 연결되는 구조를 지향합니다.
