데이터 삭제 요청 (Data Deletion Request)

앱 이름: Wefilling (위필링)

개발자: Wefilling Team

Wefilling은 사용자의 개인정보를 소중하게 생각하며, 사용자가 원할 경우 언제든지 자신의 계정과 관련된 데이터를 삭제할 수 있는 권리를 보장합니다.

1. 앱 내에서 즉시 삭제하기 (권장)

가장 빠르고 확실한 방법은 앱 내의 '회원 탈퇴' 기능을 사용하는 것입니다.

  1. Wefilling 앱을 실행하고 로그인합니다.
  2. 하단 탭에서 [내 정보] (My Page)로 이동합니다.
  3. 우측 상단의 [설정] (톱니바퀴 아이콘)을 탭합니다.
  4. [계정 설정] 메뉴를 선택합니다.
  5. [회원 탈퇴] 버튼을 누르고 안내에 따라 진행합니다.
주의: 회원 탈퇴 시 계정 정보, 게시글, 댓글, 친구 관계 등 모든 데이터가 즉시 영구적으로 삭제되며 복구할 수 없습니다.

2. 이메일로 삭제 요청하기

앱에 접속할 수 없는 경우, 아래 이메일로 데이터 삭제를 요청하실 수 있습니다.

  • 이메일: hanyangwatson@gmail.com
  • 필수 기재 사항:
    • 가입한 이메일 주소
    • 닉네임
    • 요청 내용: "계정 및 데이터 삭제 요청"

이메일 요청 시 본인 확인 절차가 진행될 수 있으며, 처리까지 최대 7일이 소요될 수 있습니다.

3. 삭제되는 데이터 유형

계정 삭제 시 다음 데이터가 영구적으로 제거됩니다:

  • 사용자 계정 정보 (이메일, UID)
  • 프로필 정보 (닉네임, 프로필 사진, 자기소개, 학교/학과 정보)
  • 작성한 게시글 및 댓글 (익명화되거나 삭제됨)
  • 친구 목록 및 차단 목록
  • 참여한 모임 내역
  • 업로드한 이미지 파일
