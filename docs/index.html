위필링 개인정보 처리방침

1. 개인정보의 처리 목적

  • 회원 가입 및 관리(본인 확인, 부정 이용 방지)
  • 서비스 제공(모임 생성·참여, 게시물·댓글 작성, 사용자 간 소통)
  • 민원 처리 및 고지

2. 개인정보의 처리 및 보유 기간

  • 회원정보: 회원 탈퇴 시까지
  • 서비스 제공 관련 기록: 서비스 이용계약 종료 시까지
  • 민원 처리 기록: 처리 종료 후 3년 보관

3. 개인정보의 제3자 제공

회사는 정보주체의 동의 또는 법령에 근거한 경우에만 개인정보를 제3자에게 제공합니다. 현재 위필링은 개인정보를 제3자에게 제공하지 않습니다.

4. 개인정보 처리의 위탁

  • 수탁자: Firebase (Google LLC)
  • 위탁 업무: 회원 인증 및 관리, 실시간 데이터 동기화, 푸시 알림 발송, 이미지 저장
  • 위탁 기간: 서비스 제공 기간 동안

5. 정보주체의 권리·의무

이용자는 다음 권리를 언제든지 행사할 수 있습니다:

  • 개인정보 열람, 정정, 삭제, 처리 정지 요구
  • 가입 해지 및 동의 철회

6. 처리하는 개인정보 항목

  • 필수: 이메일 주소, 닉네임, 학교 인증 여부 및 기록, 학과·학번(학교 인증 과정에서 수집), 프로필 사진(선택)
  • 선택: 자기소개, 게시물 콘텐츠(텍스트·이미지), 친구/팔로우 관계
  • 자동 수집: 서비스 이용 기록, 접속 로그, IP 주소, 기기 정보(OS, 단말 모델), 푸시 토큰, 앱 버전, 오류 로그

7. 개인정보의 파기

목적 달성 또는 보유 기간 만료 시 지체 없이 파기합니다.

  • 전자적 파일: 복구 불가능한 기술적 방법으로 삭제

Wefilling Privacy Policy

1. Purpose of Processing

Personal data is processed to:

  • Verify and manage user accounts, prevent misuse
  • Provide core features such as meetups, posts, comments, and messaging
  • Handle inquiries, notifications, and dispute resolution

2. Retention Period

We retain personal data only for the duration required by law or user consent:

  • User information: Until account deletion
  • Service records: Until end of service contract
  • Inquiry records: Three years after resolution

3. Third-Party Provision

Wefilling does not share personal data with third parties except with user consent or when required by law.

4. Outsourcing

  • Contractor: Firebase (Google LLC)
  • Services: User authentication, real-time data sync, push notifications, image storage
  • Period: During service provision

All outsourcing agreements include confidentiality, security, and oversight obligations as required by law.

5. User Rights

Users can exercise the following rights at any time:

  • Access, correction, deletion, and processing suspension of personal data
  • Account deletion and consent withdrawal

6. Data We Collect

  • Required: Email address, nickname, school verification status, department/student ID (collected during verification), profile photo (optional)
  • Optional: Bio, post content (text/images), friend/follow relationships
  • Automatically collected: Usage logs, access logs, IP address, device info (OS, model), push tokens, app version, error logs

7. Data Deletion

Data is deleted without delay when purposes are achieved or retention periods expire:

  • Electronic files: Deleted using technically irreversible methods
  • Physical documents: Shredded or incinerated

8. Security Measures

  • Administrative: Internal management plans, regular security training, minimal access rights
  • Technical: Access control, encryption, security programs, multi-factor authentication
  • Physical: Server access control, CCTV, entry control

9. Privacy Officer

  • Name/Department: Wefilling Development Team
  • Email: support@wefilling.com

10. Remedies

For privacy violations, contact:

  • Personal Information Protection Commission (privacy.go.kr / 118)
  • Privacy Dispute Mediation Committee (kopico.go.kr / 1833-6972)
  • Supreme Prosecutors' Office Cyber Crime Investigation Unit (spo.go.kr / 1301)
  • National Police Agency Cyber Crime Report System (police.go.kr / 182)

For international users, contact the above Korean authorities or your local data protection agency.
